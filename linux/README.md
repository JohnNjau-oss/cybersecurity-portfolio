## 📘 Topics Covered

### 1. Basic Linux Commands

* Navigation: `ls`, `cd`, `pwd`
* File operations: `touch`, `cp`, `mv`, `rm`
* Viewing files: `cat`, `less`, `head`, `tail`
* Searching: `find`, `grep`

📸 Example:
![Basic Commands](images/basic-commands.png)

---

### 2. File Permissions & Ownership

* Understanding permission format (`-rw-r--r--`)
* Changing permissions: `chmod`
* Changing ownership: `chown`

📸 Example:
![File Permissions](images/file-permissions.png)

---

### 3. Users & Groups

* Creating users: `useradd`
* Modifying users: `usermod`
* Setting passwords: `passwd`
* Understanding `/etc/passwd` and `/etc/shadow`

📸 Example:
![User Management](images/user-management.png)

---

## 🧠 Key Learnings

* Linux is permission-based, and security starts at the file level
* The terminal provides powerful control over the system
* Understanding users and privileges is critical in cybersecurity
* Hands-on practice is essential for mastering Linux

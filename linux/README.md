# Linux Fundamentals

## Commands

### ls
Lists files and directories.

Example:
ls
ls -la
ls -l

### cd (change directory)
Changes directory.

Example:
cd /home
cd ..

### pwd (print working directory)
Shows current directory.

### mkdir (make directory)
Used to create a directory

![First Commands](First-commands-cd-ls.png)

### cat (concatenate)
 Used to view items inside a file

### nano
Used to add or edit content in a file

### mv ~/source/*.file destination/
Used to move files from one directory to another

### whoami
Used to check who is the current user

## Notes
Learning how to navigate and interact with the Linux file system, which is essential for cybersecurity.
